# 🎮 Apocalite 

[![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)](https://www.python.org/)
[![Kivy](https://img.shields.io/badge/Kivy-2.3+-gray.svg)](https://kivy.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

**Apocalite** เป็นโครงการเกมแนว **Top Down Survival Roguelike** สุดมันส์ โฟกัสการต่อสู้แบบ **fast-paced, action-packed** สร้างด้วยเฟรมเวิร์ก **Kivy** ของภาษา Python โดยผู้เล่นจะต้องเอาชีวิตรอดจาก Wave ของศัตรูที่ทวีความโหดขึ้นเรื่อยๆ เผชิญหน้ากับบอสสุดแกร่ง รวมถึงการเก็บเเลเวลเพื่อปลดล็อกสกิลเทพๆ และเอาชีวิตรอดให้นานที่สุด

---

## ✨ Features (จุดเด่นของเกม)

### ⚔️ Core Gameplay (ระบบการเล่นหลัก)
- **Wave-based Survival**: สู้กับคลื่นศัตรูที่จะทวีความเก่ง จำนวน และความท้าทายขึ้นในระดับที่สูงขึ้น
- **Character Selection**: มีตัวละครให้เลือกหลากหลายที่มีสเตตัสเริ่มต้นต่างกัน (เช่น Lostman, Monkey, PTae)
- **Dynamic Skill System**: ระบบอัปเลเวลแบบสุ่มการ์ดให้ผู้เล่นเลือกอัปเกรดสกิลหรือสเตตัส (Roguelike elements)
- **Boss Fights**: ปะทะบอสสุดโหดทุกๆ 5 Wave และเตรียมรับมือกับ **Big Boss** ระดับตำนานทุกๆ 10 Wave
- **Smooth Camera Action**: ระบบกล้องแบบไดนามิกพร้อมค่า Damping ให้ความรู้สึกสมูทในทุกการเคลื่อนไหว
- **Leaderboard**: ระบบเก็บคะแนนสูงสุด

### 🧟 Enemy Types (ประเภทศัตรูที่หลากหลาย)
- **Normal**: ศัตรูระยะประชิดธรรมดา ความสมดุล
- **Stalker**: ศัตรูที่มีความเร็วสูง หลบเก่ง แต่ HP ต่ำ และจะพยายามเข้ามาประชิดตัว
- **Ranger**: ศัตรูโจมตีระยะไกล พยายามรักษาระยะห่างจากผู้เล่น
- **Charger & Bomber**: ศัตรูที่มีรูปแบบการจู่โจมพิเศษ
- **Shielder & Sniper**: ศัตรูสายแทงค์สวมเกราะ และสายซุ่มยิงดาเมจหนัก
- **Boss & Big Boss**: ศัตรูระดับสูงที่มี HP มหาศาลและมาพร้อมลูกสมุน

### 🎨 UI & UX Design (การออกแบบและส่วนต่อประสานผู้ใช้)
- **Modern & Polished Interfaces**: ดีไซน์ UI สไตล์ pixel art และการขยับ frame ของตัวละคร
- **Character Specific Items**: ไอเท็มและอาวุธพิเศษตามตัวละคร
- **Smooth Animations**: เอฟเฟกต์ No Transition และ Fade Transition ระหว่างเปลี่ยนหน้าจอ รวมถึง Hover animations
- **HUD Display**: แสดงหลอดเลือด (HP), เลเวล, แถบประสบการณ์ (EXP), หมายเลข Wave และจำนวนศัตรูที่เหลืออย่างชัดเจน และระยะเวลารการเล่น

### 🎮 Controls & Accessibility (การควบคุมที่ครอบคลุม)
เกมรองรับการควบคุมแบบ **Full Support** ในระดับหน้าจอแอปพลิเคชัน (ทั้งการเล่นและเมนู):
- **Keyboard Support**: ควบคุมเต็มรูปแบบด้วย **WASD** (ในการเคลื่อนที่/นำทางเมนู) และ **Spacebar/Enter** (ยืนยัน/พุ่งหลบ)
- **Gamepad Support**: รองรับจอยคอนโทรลเลอร์ (Analog, D-Pad, Trigger) สำหรับเลื่อนเมนูและการเล่นเกม
- **Mouse Support**: ใช้เมาส์ในการเล็งทิศทางยิง รวมถึงคลิกเลือกสิ่งต่างๆ ในเมนูได้อย่างสะดวก

## ⚙️ Installation (การติดตั้ง)

### ความต้องการของระบบ (Prerequisites)
- **Python:** เวอร์ชัน 3.7 ขึ้นไป
- **Kivy:** เวอร์ชัน 2.3 ขึ้นไป
- **Pillow** สำหรับการจัดการรูปภาพกราฟิก

### ขั้นตอนการติดตั้งและรันเกม
```bash
# 1. โคลน Repository
git clone https://github.com/psu6810110496/Kivy-Game-Project.git
cd Kivy-Game-Project

# 2. ติดตั้ง Dependencies ที่จำเป็น
pip install -r requirements.txt

# 3. ทีนี้ก็มันส์กันได้เลย!
python APOCALITE/main.py
```

## 🕹️ วิธีเล่น (How to Play)

1. **เลือกนักสู้ของคุณ**: เริ่มเกมโดยการเลือกตัวละคร (สังเกตสเตตัส HP/ATK/SPD ที่แตกต่างกันของแต่ละตัว)
2. **เคลื่อนที่ & เอาตัวรอด**: เดินหลบฝูงศัตรู ตัวละครจะทำการ**โจมตีอัตโนมัติ**ไปยังทิศทางที่คุณเล็ง และอัพเกรดสกิลเพื่อเพิ่มความสามารถในการเอาตัวรอด
3. **สะสมประสบการณ์ (EXP)**: คอยเก็บเม็ดสีเขียว (EXP Orbs) ที่ดรอปจากศัตรูที่ตายเพื่อเก่งขึ้น หรือกำจัดศัตรูให้ได้มากที่สุดเพื่อเพิ่มประสบการณ์
4. **เลือกสายอัปเกรด**: เมื่อเลเวลอัพ! เลือกการ์ดสกิลเพื่อเพิ่มความรุนแรง หรือเลือกอัปเกรด Stat ให้ทนทานต่ออุปสรรค และเพิ่มความสามารถในการเอาตัวรอด
5. **ปราบทุก Wave**: เอาตัวรอดให้ถึง Wave สูงๆ โดยความดุสยองของศัตรูจะทวีคูณ (Scaling)

### การควบคุม (Controls)

#### ลุยศึก (In-Game)
- **WASD**: เคลื่อนที่
- **เมาส์ / Analog ขวา (Right Stick)**: เล็งทิศทางการยิง (Auto-Fire)
- **คลิกซ้าย / RT**: ยิง
- **Spacebar / LT**: พุ่งหลบ (Dash)
- **ESC / Start**: หยุดเกม (Pause)
- **F11**: สลับโหมดเต็มจอ (Fullscreen)

#### เมนู (Navigation)
- **WASD / D-Pad**: เลื่อนหาตัวเลือก
- **Spacebar / Enter / A Button**: ยืนยันคำสั่ง
- **เมาส์คลิก**: เลือกคำสั่งโดยตรง
